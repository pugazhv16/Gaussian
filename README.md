# Gaussian Elimination

## AIM:
To write a program to find the solution of a matrix using Gaussian Elimination.

## Equipments Required:
1. Hardware – PCs
2. Anaconda – Python 3.7 Installation / Moodle-Code Runner

## Algorithm
1. Read the order of the matrix and the augmented matrix values.
2. Apply Gaussian Elimination to convert the matrix into upper triangular form using row operations.
3. Perform back substitution to find the values of unknown variables.
4. Display the solution of the given system of equations.
